# Github Favorites
[![NPM](https://img.shields.io/npm/l/react)](https://github.com/Toddynh0BR/Github-Favorites/blob/main/LICENSE) 

## About
Github Favorites is a web application that allows users to add other
Github users to a
favorites list just by entering their username.
Users can also remove users from the list whenever they want.

https://toddynh0br.github.io/Github-Favorites/

## Overview
The "Github Favorites" project's main objective is to create a web platform that offers users the ability to easily add and remove their favorite Github users.
This initiative aims to provide a friendly and efficient experience in managing favorites, making the process of maintaining a list of preferred users simpler and more accessible.

##  Key Features

- **Adding Users**: Users will be able to add the usernames of Github users they want to follow closely to their favorites list.

- **User Removal**: The platform allows users to easily remove any user from the favorites list, providing flexibility and full control over their preferences.

- **Organized View**: The favorites list is presented in a clear and organized way, making it easier to identify and manage added users.
